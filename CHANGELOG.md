# Changelog

All notable changes to the `mcp-gitingest` project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-24

### Added

- Initial release of the `mcp-gitingest` MCP server.
- Tool `ingest_repo` to turn GitHub repositories into AI-ready text digests.
- Support for branch selection, inclusion/exclusion patterns, and max file size limits.
- Integration with the `gitingest` Python library.
- Project boilerplate with `uv` for dependency management.
- Comprehensive documentation in `docs/` directory.
