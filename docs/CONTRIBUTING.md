# Contributing to mcp-gitingest

Thank you for your interest in improving `mcp-gitingest`!

## How to Contribute

1. **Bug Reports**: If you find a bug, please check if it has already been reported. If not, open an issue with details on how to reproduce it.
2. **Feature Requests**: We welcome ideas for new features! Please open an issue to discuss them.
3. **Pull Requests**:
    - Fork the repository.
    - Create a feature branch.
    - Write tests for your changes.
    - Ensure your code follows the project's style (standard Python type hints and docstrings).
    - Submit a pull request.

## Development Setup

We use `uv` for dependency management.

```bash
# Clone the repo
git clone [your-repo-url]
cd mcp-gitingest

# Install dependencies
uv sync

# Run the server
uv run python server.py
```

## Code of Conduct

Please be respectful and professional in all interactions.
